Today’s SME cyber lookout: fake Zoom/Webex installers, “ClickFix” lures, and faster ransomware

What small and medium-sized businesses should look out for today.

High Thursday 16 July 2026, 15:21 UK time
Today’s look-out: Malware via fake software installs and ‘copy/paste this fix’ scams; rapid ransomware disruption

What to look out for today

Three practical themes SMEs should treat as live risk today:

  • Trojanised Zoom/Webex apps being used to steal logins and financial assets.
  • “ClickFix” style lures: websites or messages that trick staff into copying/pasting a “fix” or running a command to “verify”, “unlock”, or “resolve an issue”.
  • Ransomware moving fast: a new ransomware group reportedly went from initial access to data theft and encryption in under 24 hours.

Why this matters to smaller businesses

  • Video meeting tools are universal in SMEs; a single bad install can compromise email, file access, and finances.
  • Finance and operations are high-impact targets: credential theft often leads to invoice fraud, payroll diversion, and supplier payment scams.
  • Speed reduces your “reaction window”. If attackers can move to encryption within a day, delays in detection, isolation, or incident response become very costly.

Warning signs

  • Staff report a link to “download Zoom/Webex” that isn’t from your normal software portal or vendor channel.
  • A browser pop-up, email, or chat message instructs the user to copy/paste text into Terminal/PowerShell to “fix” something.
  • On macOS, users see repeated prompts for their password or unusual behaviour where apps keep closing/crashing until they enter it.
  • Multiple failed logins, new device sign-ins, or unexpected MFA prompts across Microsoft 365/Google/Dropbox.
  • Sudden access issues to shared drives, backups, or cloud folders; unusual file renaming/encryption indicators.

How attackers may exploit the situation

  • Trojanised installers imitate legitimate Zoom/Webex downloads to install backdoors and steal credentials.
  • Social engineering “fixes” persuade a user to run commands themselves (bypassing some security controls) under the pretence of solving an error.
  • Rapid ransomware playbooks aim to get admin access quickly, steal data, then encrypt before defenders can respond.

What to do today

  • Send a 2-minute staff warning: “Don’t install Zoom/Webex from links in emails/chats. Don’t copy/paste ‘fix’ commands into Terminal/PowerShell. If prompted, stop and report it.”
  • Standardise software installs: use a managed software portal/MDM where possible; block staff from installing meeting software ad-hoc.
  • Harden account access: ensure MFA is on for email, file storage and accounting; review any recent unusual sign-ins.
  • Rehearse ‘first hour’ ransomware steps: who isolates devices, who contacts IT/provider, what gets shut off (and what shouldn’t).

Ask your IT provider

  • Can we restrict software installation (especially remote access and meeting apps) to approved sources only?
  • Do you have alerting for credential theft signals (impossible travel, MFA fatigue patterns, new device logins) across our key SaaS?
  • If ransomware hits and moves fast, what is your containment runbook in the first 30–60 minutes?
  • Are our backups segmented and recoverable, and when was the last restore test?

Patch watch - only one short paragraph, and only if relevant

Microsoft has warned that some Windows editions will reach end of support in about 90 days. For SMEs, the key point is planning: confirm which PCs are in scope, what the upgrade path is, and whether any line-of-business apps will be affected—don’t leave it to the last month.

One action today

Send a same-day internal note: “Only install Zoom/Webex via our approved method; never copy/paste ‘fix’ commands into Terminal/PowerShell; report any password prompts or app-crash loops immediately.”
